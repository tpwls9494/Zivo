## Chrome Web Store Listing Fields

**Name**: Zivo — 한일 항공권 최저가 & 자동완성

**Short description (max 132 chars)**:
한국-일본 노선 편도 조합 최저가 탐색 + 항공사 예약 폼 탑승자 정보 원터치 자동완성

**Description**:
Zivo는 한국↔일본 노선(ICN·GMP ↔ KIX·NRT·FUK·CTS)의 편도 조합 최저가를 찾아주고, 저장된 탑승자 정보를 항공사 예약 폼에 자동으로 입력해주는 크롬 익스텐션입니다.

✈ 편도 조합 최저가
서로 다른 항공사의 편도 티켓을 조합해 왕복권보다 저렴한 가격을 제시합니다.

🗓 달력 히트맵
주말·날짜별 최저가를 한눈에 확인하세요.

⚡ 원터치 자동완성
대한항공·아시아나·JAL·ANA·진에어·제주항공·에어부산·티웨이·피치·제트스타 예약 폼에 여권 영문명·생년월일·연락처를 자동 입력합니다.

🔒 보안
여권번호는 AES-256-GCM으로 암호화되어 서버에만 저장되며, 결제 정보는 절대 수집하지 않습니다.
